Politique de Confidentialité

Introduction

Cette politique de confidentialité décrit comment notre API GPT-Bilan Thermique gère les données des utilisateurs et les informations traitées. Votre confidentialité est une priorité.

Données Collectées

Notre API ne collecte aucune donnée personnelle. Les seules informations traitées sont celles fournies via les paramètres de requêtes nécessaires aux calculs (par exemple, valeurs physiques ou techniques).

Ces données ne sont ni stockées ni partagées. Elles sont uniquement utilisées temporairement pour produire les résultats des calculs demandés.

Utilisation des Données

Les données envoyées à notre API sont utilisées exclusivement pour exécuter les calculs demandés. Elles ne sont pas enregistrées dans des bases de données ni transmises à des tiers.

Sécurité

Notre API utilise des clés d'authentification pour limiter l'accès et assurer la sécurité des requêtes. Veuillez garder votre clé privée sécurisée.

Partage des Informations

Aucune information traitée par notre API n'est partagée avec des tiers.

Modifications de cette Politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page.
